This website is blocked by your Cox Parental Controls settings.

The primary account holder (generally a parent or guardian) can change these settings if needed. Just follow these steps:

  1. Go to myaccount.cox.net.
  2. Log in with the primary user name and password.
  3. Click "Run Cox Parental Controls."
  4. Add the website you want to permit to the "Allow List." You can also change the device's age-appropriate level to a higher setting.

For more Cox Internet information and troubleshooting tips, visit Internet Support.
